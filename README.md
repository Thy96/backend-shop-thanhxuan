# Shop Thanh Xuan

E-commerce website built with Next.js and Node.js.

## Tech Stack

Frontend:

- Next.js
- TypeScript
- TailwindCSS

Backend:

- Node.js
- Express
- MongoDB

## Project Structure

shop-thanhxuan
├ BE
└ FE

## Installation

### Frontend

cd FE
npm install
npm run dev

### Backend

cd BE
npm install
npm start
